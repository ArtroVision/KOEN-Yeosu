# KOEN-Yeosu

한국남동발전 여수발전본부의 SMART LOTO 키 보관함 키오스크 WPF 앱 개발 리포지토리입니다.

## Prototype Page

첨부 프로토타입/콘셉트 이미지는 GitHub Pages에서 볼 수 있도록 `docs/` 정적 웹 페이지로 구성했습니다.

- 로컬 확인: `docs/index.html`을 브라우저로 열기
- 배포 주소: <https://artrovision.github.io/KOEN-Yeosu/>

GitHub Actions의 `Deploy GitHub Pages` 워크플로가 `main` 브랜치에 push될 때 `docs/` 폴더를 GitHub Pages로 배포합니다.
