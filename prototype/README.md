# Prototype

첨부받은 웹 프로토타입 및 참고 이미지를 WPF 본 앱과 분리해서 관리하는 폴더입니다.

- `web/`: HTML/CSS/JavaScript 기반 키오스크 화면 프로토타입
- `attachments/`: 전달받은 첨부 원본 이미지
- `assets/`: 프로토타입/문서용 참고 이미지
- `docs/`: 이전 GitHub Pages용 정적 문서 백업
- 실행: `web/index.html`을 브라우저로 열기

> 이 폴더의 파일은 WPF 구현을 위한 참고/시안이며, 실제 운영 앱의 메인 코드는 `src/KOENYeosu.Kiosk/`에 둡니다.
